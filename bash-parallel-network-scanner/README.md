# Parallel Network Scanner
This repository contains a Bash script built for automated network scanning with parallel processing.

Inside you’ll find the main script `parallel-network-scanner.sh` along with supporting files that make it run. The script reads a list of targets from a file, validates the input, and processes multiple targets at the same time to generate reports. Each run saves results to timestamped folders so outputs stay organized.

The project is structured to be simple and self-contained. You’ll see the core scanning logic in the script, a file for listing targets, and folders where reports are stored after execution. Sample data is included so you can test it without needing external systems.

This repo demonstrates practical Bash scripting, input validation, file handling, and parallel execution for security automation tasks.
